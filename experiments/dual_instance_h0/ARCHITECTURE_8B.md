## 5. 数据中心扇出架构 (h^(0) 的真正价值)

```
                         ┌──────────────────┐
                         │   Instance A     │
                         │   (Embedder)     │
                         │   631 MB         │
                         │                  │
                         │  Prompt → h^(0)  │
                         │  ~0ms            │
                         └────────┬─────────┘
                                  │
                     h^(0) = 16 MB (2K ctx)
                     (vs Full KV = 288 MB)
                                  │
                 ┌────────────────┼────────────────┐
                 │                │                │
                 ▼                ▼                ▼
        ┌──────────────┐ ┌──────────────┐ ┌──────────────┐
        │ Instance B₁  │ │ Instance B₂  │ │ Instance B₃  │
        │ (Decoder)    │ │ (Decoder)    │ │ (Decoder)    │
        │ 8.3 GB       │ │ 8.3 GB       │ │ 8.3 GB       │
        │              │ │              │ │              │
        │ Recon → KV   │ │ Recon → KV   │ │ Recon → KV   │
        │ → Decode     │ │ → Decode     │ │ → Decode     │
        │ 24 t/s       │ │ 24 t/s       │ │ 24 t/s       │
        └──────────────┘ └──────────────┘ └──────────────┘

        ✦ 1 个 A 服务 N 个 B → Prefill 成本分摊 N×
        ✦ h^(0) 可缓存 → 相同 prompt 跳过所有 Prefill
        ✦ A/B 独立扩缩容 → 按需调度
```
